How it works

Three steps. No spreadsheets, no onboarding call, no panic.

01

Add the bot

Invite AdminVU to your server with the usual Discord permissions. It boots up, says hi, and starts watching — that's it.

02

Set up your servers

Open the local control panel, pick which servers opt into propagation, set your notification channels, and you're configured.

03

We handle the rest

Warnings get logged, bad actors get propagated, tickets get tracked, members get trivia. You get to relax.
